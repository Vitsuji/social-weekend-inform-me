## Social Weekend
### InformMe

> The people who lost those objects found can easily retrieve them in a quick manner.

### Installation for dev

Install the dependencies and devDependencies and start the server.

```sh
$ cd social-weekend-inform-me
$ npm install
$ npm run dev
```
